# Zomato Restaurant Analysis

## Project Overview
This project analyzes Zomato restaurant data to identify trends in ratings, cuisines, costs, and customer preferences.

## Tools Used
- Excel
- SQL
- Tableau
- Power BI

## Key Insights
- Top-rated restaurants
- Popular cuisines
- Cost distribution
- Online ordering trends
